# 谛深禅诗数字策展

本语境描述谛深诗作从权威文本进入公开数字策展体验时使用的核心语言。

## 作品状态

**权威原文（authoritative text）**：
经指定来源确认的诗题、正文、标点、特殊用字与创作日期；网站和 AI 制作流程都不得凭推测修订它。
_避免_：清理后的正文、AI 修正版

**显示派生文本（display-derived text）**：
从权威原文通过受控简繁转换与人工例外词生成、只用于界面显示的文本；它不能独立编辑，也不取代权威原文。
_避免_：译文、第二版正文

**作品地址（poem address）**：
作品首次公开时确定、此后不随标题修正或界面语言切换而变化的访问地址；它由创作日期与标题拼音构成的人类可读 slug 标识。
_避免_：文件路径、当前标题

**已校勘作品（verified poem）**：
原文与创作日期已经人工确认、可以公开阅读，但尚未完成全部视觉、声音、权利与多端预览审核的作品。
_避免_：已策展、已完成

**人工校勘（human verification）**：
由有权确认来源的人逐项核对诗题、正文、标点、特殊用字、创作日期与来源，并决定作品能否从“已收录”晋级为“已校勘”。AI 不具有这项决定权。
_避免_：自动清理、AI 审核

**诗境待成（visual curation pending）**：
面向访客说明作品文字已经校勘、最终视觉与声音策展尚未完成的克制状态；它不是阻止阅读的“施工中”页面。
_避免_：已策展、网站未完成

**已策展作品（curated poem）**：
已校勘，并且图片、声音、署名、许可证、无障碍描述和多端体验均通过人工审核，可以进入沉浸式诗境的作品。
_避免_：仅因页面可访问就称为已策展

## 交付范围

**展览（exhibition）**：
经过整体策划的一组有序作品；入选关系和阅读顺序属于展览本身，不属于单首诗。
_避免_：精选标记、作品自身顺序

**当前诗境（current exhibition）**：
网站此刻公开呈现的唯一已发布展览；新方案在草稿审核期间不改变它。
_避免_：最新作品、草稿展览

**展览草稿（draft exhibition）**：
等待整体审核的候选作品与顺序清单，不进入公开网站。
_避免_：当前诗境、已发布展览

**MVP 公众网站**：
可在本地运行并可部署的首版公开网站，提供五首诗的长卷入口、稳定单诗页面、档案骨架和无障碍阅读路径；未获审核的图片、声音或作者资料不得伪装成最终素材。
_避免_：最终策展成品、单个演示 HTML

**程序化水墨场景**：
在最终策展图片尚未获批时，由网页样式与图形构成的明确可替换视觉层；它服务于布局和氛围验证，不使作品升级为“已策展”。
_避免_：最终主视觉、已批准图片

**近日诗卷（latest five）**：
按 `writtenDate DESC, id ASC` 从全部公开作品自动选出的最多五首，是首页的阅读入口；它不是展览，也不保存人工顺序。
_避免_：当前诗境、近日新作卡片

**视觉档案（visual profile）**：
作品可选的色谱、意象、构图、光线与强度策展字段；省略值可由确定性规则补齐。正文自动识别出的视觉信号是内部实现信息，不等同于公开描述标签。
_避免_：固定五阶段、自动标签

**PDF 版式模板（PDF layout template）**：
由标题、正文和日期的字号、区域与结构特征形成的抽取规则签名；每种模板必须有一首代表作通过人工校准，才可用于高置信自动公开。
_避免_：某一页、导入批次

**已收录候选（ingested candidate）**：
已保存来源与抽取审计信息、但尚未满足公开条件的诗歌候选；未校准版式、低置信结果和未启用年份都保持隐藏。
_避免_：已校勘作品、公开草稿

**已审核声景（approved soundscape）**：
来源、作者、许可证、使用方式和播放效果均通过人工审核的音乐或自然环境声组合；作品只有引用此类资源时才显示声音入口与播放控件。
_避免_：占位音频、无声播放器

**候选音频（candidate audio）**：
已核对来源元数据并下载到本地、但尚未通过用户试听批准的古琴或环境声；候选只出现在开发试听页，不能进入正式播放器或公开署名清单。
_避免_：已审核声景、默认背景音乐

**点缀声（accent audio）**：
默认关闭、非循环、稀疏调度的竹笛、颂钵或风铃。点缀出现时临时降低主音乐与环境层有效音量，但不得改写用户保存的基础音量。点缀声与连续环境声是不同领域角色。
_避免_：循环环境声、第二条背景音乐
