# 增加稀疏点缀层与音量避让

正式声景在 ADR 0005 的古琴主层和单选环境层之上增加非循环点缀层。点缀只可选择竹笛、颂钵、风铃或混合模式，默认关闭，同一时刻最多播放一个素材。

点缀开始前以 600 ms 将主层有效音量乘以 `0.45`、环境层乘以 `0.71`，结束后以 1,800 ms 恢复；用户保存的基础音量不被改写。首次点缀等待 45–90 秒，之后间隔 120–240 秒。实现继续使用持久化原生音频元素与 `requestAnimationFrame`，本阶段不引入 Web Audio。

七项新增素材得到站点所有者直接批准，但仍必须通过许可元数据、解码、文件尺寸、响度、峰值与 SHA-256 检查。旧有七项候选不因本决策自动批准。
