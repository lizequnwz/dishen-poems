# 谛深禅诗 MVP 实施计划

依据 [网站设计规格](../specs/2026-07-20-dishen-zen-poetry-website-design.md)、根目录 `CONTEXT.md` 与 `docs/adr/` 中已确认决策实施。

## 1. 内容基础

- 初始化 Astro 静态站点与 TypeScript 严格模式。
- 将五首诗迁移为带 frontmatter 的单文件 Markdown，并按日期与诗题改名。
- 建立诗歌与展览 collection schema、跨记录校验和简繁派生层。
- 建立首个已发布展览清单，保留草稿展览的本地预览边界。

## 2. 公众页面

- 首页：入口、五幕纵向长卷、近日新作和三个展览出口。
- 单诗页：稳定 URL、权威原文、简繁切换、上下首、分享与状态说明。
- 档案与年份页：新到旧、年份跳转、最新、最早和随机漫游。
- 关于页：照片与简介占位、官方网站链接。
- 署名页：明确当前没有外部策展媒体，并说明未来权利记录要求。

## 3. 视觉与交互

- 建立宣纸晨雾／夜山墨海 token 和每诗季节色接口。
- 用装饰性 CSS/SVG 场景实现一实一虚、多留白的可替换水墨层。
- 实现中英文 UI、原／简／繁、主题偏好、分享与随机漫游。
- 保证无 JavaScript 阅读路径、键盘、焦点、44px 触控、Reduced Motion 和资源失败降级。

## 4. 验证与交付

- 内容、路由、展览、转换与公开边界自动测试。
- Astro check、生产构建和输出检查。
- 在 375、768、1024、1440px 及 Light/Dark 关键组合进行浏览器截图检查。
- 将未完成的真实素材、分析、审核工具和 AI 制作台写入系统临时目录的 handoff。
